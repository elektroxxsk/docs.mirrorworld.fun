# `recommendSearchNFT`
Get recommended NFTs from a collection. This API will provide 10
NFT info at most as recommend NFT. Developer can use this API to fill some recommend blank when user trying search.

```java
List<String> collections = new ArrayList<>();
collections.add("collection address 1");

MWEVM.recommendSearchNFT(collections, new MirrorCallback() {
    @Override
    public void callback(String result) {
        //todo
    }
});
```
