# `openWallet`
Open wallet page. The param is a `callback which will be called when the user clicks the "Logout" button on the wallet page.

```java
MirrorWorld.openWallet(new MirrorCallback() {
    @Override
    public void callback(String result) {
        Log.d("MirrorSDK","User logout from wallet page.");
    }
});
```
