# `getCollectionsInfo`
Get collections info by collection addresses.

```swift
MWSDK.Solana.Metadata.getCollectionsInfo(collections: [collection_mint]) {data in
    print(data)
} onFailed: {code, message in
    print("failed  code:\(code),message: \(message ?? "")")
}
```
