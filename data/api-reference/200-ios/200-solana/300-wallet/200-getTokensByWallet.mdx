# `getTokensByWallet`

Allow users to check wallet tokens by a wallet address.
```Swift
MWSDK.Solana.Wallet.getTokensByWallet(wallet_address: to_wallet_address) { response in
    print("success, result is:\(String(describing: response))")
} onFailed: {
    print("failed ~")
}
```
