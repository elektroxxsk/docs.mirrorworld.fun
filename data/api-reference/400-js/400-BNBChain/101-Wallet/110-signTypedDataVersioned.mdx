# `signTypedDataVersioned`

Performs Off Chain Versioned Signing Of An Arbitrary Typed Message

```js
import { MirrorWorld, BNBChain } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: BNBChain("bnb-mainnet"),
})

await mirrorworld.BNBChain.Wallet.signTypedDataVersioned()
```
