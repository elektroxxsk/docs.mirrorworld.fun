# `fetchTokensInWallet`

Get The Erc Tokens Inside An BNB Chain Wallet Address

```ts
const wallet = "0x137960064dA83737d3A761E9A239EF33887cdAf4"
await mirrorworld.BNBChain.Wallet.fetchTokensInWallet(wallet)
```
