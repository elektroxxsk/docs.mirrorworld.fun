# `signTypedData`

Performs Off Chain Signing Of A An Arbitrary Typed Message

```js
const payload = {
  data: [
    {
      name: "data",
      type: "string",
      value: "hello world",
    },
  ],
}
await mirrorworld.BNBChain.Wallet.signTypedData(payload)
```
