# `fetchWalletTransactions`

Fetch BNB Chain Transactions In Wallet Address

```js
const wallet = "0x29325cE945EB4051265d8d1E8A97C7b52524E602"

await mirrorworld.BNBChain.Wallet.fetchWalletTransactions(wallet)
```
