# `fetchTransactionByHash`

Fetch Bnb Chain Transaction Data By Transaction Hash

```js
import { MirrorWorld, BNBChain } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: BNBChain("bnb-mainnet"),
})

await mirrorworld.BNBChain.Wallet.fetchTransactionByHash()
```
