# `fetchTransactionByHash`

Fetch BNB Chain Transaction Data By Transaction Hash

```js
const hash =
  "0x5a7712e4340fbf761521152ca44abfec786aa69cc447d9e936a362c0587f2037"

await mirrorworld.BNBChain.Wallet.fetchTransactionByHash(hash)
```
