# `searchNFTs`

Search Bnb Chain Nf Ts By Their Contract Addresses And Token Ids

```js
import { MirrorWorld, BNBChain } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: BNBChain("bnb-mainnet"),
})

await mirrorworld.BNBChain.Asset.searchNFTs()
```
