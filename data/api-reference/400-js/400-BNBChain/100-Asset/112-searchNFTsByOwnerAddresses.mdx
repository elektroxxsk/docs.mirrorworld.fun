# `searchNFTsByOwnerAddresses`

Searches Bnb Chain NFTs By Owner Addresses

```js
import { MirrorWorld, BNBChain } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: BNBChain("bnb-mainnet"),
})

await mirrorworld.BNBChain.Asset.searchNFTsByOwnerAddresses()
```
