# `updateMarketplace`

Updates A Sn Existing NFT Marketplace On Bnb Chain

```js
import { MirrorWorld, BNBChain } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: BNBChain("bnb-mainnet"),
})

await mirrorworld.BNBChain.Asset.updateMarketplace()
```
