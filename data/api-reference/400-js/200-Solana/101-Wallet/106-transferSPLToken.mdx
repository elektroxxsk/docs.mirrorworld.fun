# `transferSPLToken`

Transfers Spl Tokens From User S Wallet Address To Another Wallet Address

```js
import { MirrorWorld, Solana } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Solana("mainnet"),
})

await mirrorworld.Solana.Wallet.transferSPLToken()
```
