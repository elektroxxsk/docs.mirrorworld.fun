# `fetchTransactionBySignature`

Get Raw Transaction Data From Signature

```ts
const signature =
  "5VaXx8S3EAU4yhZ81u6RVKdJmT7wJ5z4cQkxR381XdJLm5Dh7qhEFZNRrzVHeD9UUYdzrCrSEZZT3bn7Q1ouyVyh"

await mirrorworld.Solana.Wallet.fetchTransactionBySignature(signature)
```
