# `fetchTransactions`

Get Users Transactions On Solana Requires Authentication

```js
import { MirrorWorld, Solana } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Solana("mainnet"),
})

await mirrorworld.Solana.Wallet.fetchTransactions()
```
