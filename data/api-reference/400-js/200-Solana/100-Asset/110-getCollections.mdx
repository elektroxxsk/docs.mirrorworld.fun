# `getCollections`

Creates A New Verified Collection On Solana

```js
import { MirrorWorld, Solana } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Solana("mainnet-beta"),
})

await mirrorworld.Solana.Asset.getCollections()
```
