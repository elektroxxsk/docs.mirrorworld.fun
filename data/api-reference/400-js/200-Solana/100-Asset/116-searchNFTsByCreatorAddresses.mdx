# `searchNFTsByCreatorAddresses`

Searches Solana Nf Ts By Creator Addresses

```js
import { MirrorWorld, Solana } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Solana("mainnet-beta"),
})

await mirrorworld.Solana.Asset.searchNFTsByCreatorAddresses()
```
