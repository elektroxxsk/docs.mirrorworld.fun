# `searchNFTs`

Search Polygon NFTs By Their Contract Addresses And Token Ids

```js
import { MirrorWorld, Polygon } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Polygon("mumbai-mainnet"),
})

await mirrorworld.Polygon.Asset.searchNFTs()
```
