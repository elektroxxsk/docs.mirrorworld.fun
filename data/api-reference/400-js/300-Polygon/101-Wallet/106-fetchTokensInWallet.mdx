# `fetchTokensInWallet`

Get The Erc Tokens Inside An Polygon Wallet Address

```js
import { MirrorWorld, Polygon } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Polygon("mumbai-mainnet"),
})

await mirrorworld.Polygon.Wallet.fetchTokensInWallet()
```
