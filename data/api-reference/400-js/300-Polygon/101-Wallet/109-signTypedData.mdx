# `signTypedData`

Performs Off Chain Signing Of A An Arbitrary Typed Message

```js
import { MirrorWorld, Polygon } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Polygon("mumbai-mainnet"),
})

await mirrorworld.Polygon.Wallet.signTypedData()
```
