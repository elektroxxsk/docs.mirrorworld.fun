# `fetchTransactionsByWallet`

Get The Transactions Authored By An Polygon Wallet Address

```js
const wallet = "0x137960064dA83737d3A761E9A239EF33887cdAf4"
await mirrorworld.Polygon.Wallet.fetchTransactionsByWallet(wallet)
```
