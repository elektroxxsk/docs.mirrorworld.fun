# `fetchTransactionsByWallet`

Get The Transactions Authored By An Polygon Wallet Address

```js
import { MirrorWorld, Polygon } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Polygon("mumbai-mainnet"),
})

await mirrorworld.Polygon.Wallet.fetchTransactionsByWallet()
```
