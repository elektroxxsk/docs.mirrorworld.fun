# `fetchNFTsInfo`

Fetch Polygon NFTs Info Accepts Filtering Metadata

```js
import { MirrorWorld, Polygon } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Polygon("mumbai-mainnet"),
})

await mirrorworld.Polygon.Metadata.fetchNFTsInfo()
```
