# `fetchTokensInWallet`

Get The Erc Tokens Inside An Ethereum Wallet Address

```js
import { MirrorWorld, Ethereum } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Ethereum("mainnet"),
})

await mirrorworld.Ethereum.Wallet.fetchTokensInWallet()
```
