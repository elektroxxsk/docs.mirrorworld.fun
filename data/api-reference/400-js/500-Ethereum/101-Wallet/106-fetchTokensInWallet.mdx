# `fetchTokensInWallet`

Get The ERC20 Tokens Inside An Ethereum Wallet Address

```js
const wallet = "0x137960064dA83737d3A761E9A239EF33887cdAf4"
await mirrorworld.Polygon.Wallet.fetchTokensInWallet(wallet)
```
