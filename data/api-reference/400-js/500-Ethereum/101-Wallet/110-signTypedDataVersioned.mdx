# `signTypedDataVersioned`

Performs Off Chain Versioned Signing Of An Arbitrary Typed Message

```js
import { MirrorWorld, Ethereum } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Ethereum("mainnet"),
})

await mirrorworld.Ethereum.Wallet.signTypedDataVersioned()
```
