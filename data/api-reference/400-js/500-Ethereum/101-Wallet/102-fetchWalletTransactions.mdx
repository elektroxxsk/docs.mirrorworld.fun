# `fetchWalletTransactions`

Fetch Ethereum Transactions In Wallet Address

```js
const wallet = "0x29325cE945EB4051265d8d1E8A97C7b52524E602"

await mirrorworld.Ethereum.Wallet.fetchWalletTransactions(wallet)
```
