# `fetchTransactionsByWallet`

Get The Transactions Authored By An Ethereum Wallet Address

```js
const wallet = "0x137960064dA83737d3A761E9A239EF33887cdAf4"
await mirrorworld.Ethereum.Wallet.fetchTransactionsByWallet(wallet)
```
