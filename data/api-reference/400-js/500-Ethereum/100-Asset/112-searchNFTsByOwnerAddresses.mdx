# `searchNFTsByOwnerAddresses`

Searches Ethereum NFTs By Owner Addresses

```js
import { MirrorWorld, Ethereum } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Ethereum("mainnet"),
})

await mirrorworld.Ethereum.Asset.searchNFTsByOwnerAddresses()
```
