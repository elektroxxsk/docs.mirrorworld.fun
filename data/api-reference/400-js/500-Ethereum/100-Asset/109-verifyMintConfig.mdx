# `verifyMintConfig`

Verifies The Mint Configuration Of A Ethereum NFT

```js
import { MirrorWorld, Ethereum } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Ethereum("mainnet"),
})

await mirrorworld.Ethereum.Asset.verifyMintConfig()
```
