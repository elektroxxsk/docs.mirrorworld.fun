# `registerCollection`

Register An Nft Collection Info To Metadata Service After Successful Registration The Metadata Service Will Index The Collection Data

```js
import { MirrorWorld, Ethereum } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Ethereum("mainnet"),
})

await mirrorworld.Ethereum.Metadata.registerCollection()
```
