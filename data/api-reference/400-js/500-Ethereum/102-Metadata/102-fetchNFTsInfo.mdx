# `fetchNFTsInfo`

Fetch Ethereum Nf Ts Info Accepts Filtering Metadata

```js
import { MirrorWorld, Ethereum } from "@usemirrorworld/web3.js"

const mirrorworld = new MirrorWorld({
  apiKey: "YOUR_API_KEY",
  chainConfig: Ethereum("mainnet"),
})

await mirrorworld.Ethereum.Metadata.fetchNFTsInfo()
```
