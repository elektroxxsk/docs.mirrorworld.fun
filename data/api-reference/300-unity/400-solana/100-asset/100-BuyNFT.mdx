# `BuyNFT`

Purchase NFT on specified Marketplace.
### Parameters
`mintAddress|string` - Mint address of a NFT.

`price|double` - Price of this NFT on marketplace.

`auction_house|string` - Which marketplace the target NFT on.

### Returns
`response class` - `CommonResponse<SolResBuyNFT>`

### Code Example
```csharp
string mintAddress = "NFT mint address on solana";
double price = "NFT's price";
string auction_house = "auction house which NFT is on";

MWSDK.Solana.Asset.BuyNFT(mintAddress, price, auction_house, null, (res) => {
    CommonResponse<SolResBuyNFT> resObj = res;
    long http_status_code = res.http_status_code;
    SolResBuyNFTNft nft = res.data.nft;

});
```
