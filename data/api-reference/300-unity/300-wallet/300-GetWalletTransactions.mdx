# `GetWalletTransactions`

Get the transactions of user's wallet.

```csharp
float price = 0.1f;
string nextBefore = "nextBefore";
MirrorSDK.GetWalletTransactions(price,nextBefore,(res)=> {
    var body = JsonUtility.ToJson(res);
    Debug.Log("result:" + body);
});
```
