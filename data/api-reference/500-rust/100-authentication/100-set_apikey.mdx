# `set_apikey`
Sets the API key of your project to the SDK instance. This can be acquired from
the Mirror World [Developer Dashboard](/overview/developer-dashboard)

```rust
use mirrorworld_sdk_rust::{ set_apikey };

fn main() {
    // set global apikey
    set_apikey("SUPER_SECRET_API_KEY");
}
```
