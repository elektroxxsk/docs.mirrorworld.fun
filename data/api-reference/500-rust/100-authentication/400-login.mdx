# `login`
Logs in a user with email and password.

```rust
use mirrorworld_sdk_rust::{ login };

pub struct LoginParam  <'a>{
  pub email: &'a str,
  pub password: &'a str,
}

fn main() {
    let payload = LoginParam {
        email: "email@example.com",
        password: "super-safe-password"
    };
    let res = login(payload);
}
```
